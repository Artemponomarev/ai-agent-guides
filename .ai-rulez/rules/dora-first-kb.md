# Dora-First KB Retrieval

- For questions about AI agent platforms, skills, tools, installation, or safety, search the KB with Dora before answering.
- Treat `guides/` as the canonical source of truth.
- Prefer `dora` in this workspace.
- Run Dora searches sequentially when possible; the local Dora DB can lock under parallel access.
- After KB updates, refresh Dora with `dora index --skip-scip`.
