# AI Agent Guides

A knowledge base for AI agent skills, tools, and platforms. Used as a reference for me and my agents.

## Structure

- `guides/` — Platform-specific guides for discovering and installing AI agent skills
- `resources/` — Links, references, and curated lists

## Indexing

This repo uses [Dora](https://github.com/dora-rs/dora) for indexing:

```bash
dora index
```
